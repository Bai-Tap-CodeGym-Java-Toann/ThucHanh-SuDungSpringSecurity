# LanaBaseGradle
### Lana's Project Gradle - LTS

How To Use:
1. Clone 
2. Delete .git folder 
3. Rename folder
4. Rename rootProject name in settings.gradle
5. Build project

Remember To:
* Re config files in configuration package (SpringDataJPA)
* Create package: 
  - controllers
  - repository
  - service
  - model
